Real colspans

To make sure the table still makes semantic sense (and sense for screen reader users), we double up the column headers, then visually hide one of them.

We need to be careful if we set display on a column header th. This can change its implicit role, breaking the table semantics! We would need to add role="columnheader" so that the table keeps its semantics.
