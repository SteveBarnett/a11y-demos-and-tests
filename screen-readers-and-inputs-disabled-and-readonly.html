Screen readers and inputs, disabled and readonly

Note: some content on this page has had aria-hidden="true" for more precise testing.

Not disabled, not readonly.

This sounds something like "First name, edit text"

When you type printable characters (letters, punctuation, space) you hear the characters.

Set to disabled.

This doesn't receive keyboard focus! But in reading mode, this sounds something like "Middle name, dimmed, edit text"

Set to readonly.

This sounds something like "Last name, edit text"

Note how it doesn't mention the readonly status. However, when you type printable characters (letters, punctuation, space) you don't hear the characters. So it is possible to infer that nothing is happening...
